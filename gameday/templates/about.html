About

This site lets you know if there is a major sporting event at one of the stadiums near downtown Atlanta so you can plan around any potential traffic jams.

I was inspired by isthereagametoday.com.

The following events are currently included:

  • Braves games
  • Falcons games
  • Hawks games
  • Georgia Tech football
  • GSU football
  • Other selected Dome/Philips Arena events

I built it as a way to learn Django development/deployment. Is a PostgreSQL database with one table overkill? Absolutely. But I had to start somewhere to learn this stuff.

The code is on GitHub.
